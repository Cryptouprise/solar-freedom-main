# Solar Freedom — Design Brainstorm

## Selected Design: **"The Reckoning"** — Dark Industrial Fury

### Design Movement
Industrial Brutalism meets Cinematic Legal Drama. Think: a law firm that also fights wars. Dark, commanding, urgent.

### Core Principles
1. **Psychological Urgency** — Every element creates a sense that time is running out and action is required NOW
2. **Asymmetric Power** — Off-center layouts, diagonal cuts, and bold weight contrasts signal dominance and confidence
3. **Earned Trust** — Real numbers, real faces, real proof — no vague claims, everything is specific and credible
4. **Controlled Aggression** — The design is angry on behalf of the visitor. It validates their frustration.

### Color Philosophy
- **Background**: Near-black charcoal `#0D0F14` — creates cinematic weight and authority
- **Primary Accent**: Burning amber/orange `#F97316` — urgency, fire, action (contrasts powerfully with dark bg)
- **Secondary Accent**: Electric white `#F8FAFC` — clarity, truth, legal precision
- **Danger Red**: `#EF4444` — used sparingly for pain-point callouts
- **Muted Surface**: `#1A1D26` — card backgrounds, slightly lifted from base

### Layout Paradigm
- Full-bleed dark hero with diagonal text cut and particle/glow overlay
- Asymmetric two-column layouts throughout (never centered boring grids)
- Sticky side-anchored CTA that follows the user down the page
- Diagonal section dividers (clip-path) to create visual momentum

### Signature Elements
1. **Glowing amber number counters** — animated stat counters (e.g., "2,847 contracts reviewed")
2. **Broken chain / freedom motif** — subtle icon language throughout
3. **Red "TRAPPED" vs green "FREE" contrast labels** — used in testimonials and process steps

### Interaction Philosophy
- Scroll-triggered entrance animations (fade-up + slight scale)
- Hover states that feel like "activating" something — glow pulses, border reveals
- AI chat widget that slides in from bottom-right with a pulsing indicator

### Animation
- Hero: slow-zoom background parallax + floating particle dust
- Stats: counter roll-up on scroll enter
- Cards: staggered fade-up with 80ms delays
- CTA buttons: amber glow pulse on idle, scale + brightness on hover

### Typography System
- **Display/Hero**: `Bebas Neue` — tall, aggressive, commanding (all-caps headlines)
- **Body/Sub**: `DM Sans` — clean, modern, readable (not Inter)
- **Accent/Legal**: `DM Mono` — used for numbers, stats, legal fine print
- Hierarchy: 96px hero → 48px section → 28px card title → 16px body
